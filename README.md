# tutorista
================

## Project Overview

The tutorista project is a web application developed using JavaScript, CSS, and HTML, with React and Vite as the primary frameworks and libraries.

## Installation

To install and run the project, follow these steps:

1. Clone the repository from GitHub: https://github.com/EshaalB/tutorista
2. Navigate to the project directory
3. Run the following command to start the development server: `npm run dev`
4. Run the following command to build the project: `npm run build`

## Usage

The project provides several npm scripts for development and deployment:

* `dev`: Starts the development server using Vite
* `build`: Builds the project using Vite
* `lint`: Runs ESLint on the project code
* `preview`: Starts a preview server using Vite

## Project Structure

The project structure is as follows:

* `.gitignore`: Configuration file for Git
* `eslint.config.js`: ESLint configuration file
* `index.html`: Main HTML file
* `package-lock.json`: Lockfile for npm packages
* `package.json`: Project metadata and dependencies
* `public/`: Public assets directory
 * `founders/`: Directory for founder images
 * `logo.png`: Project logo
 * `vite.svg`: Vite logo
* `src/`: Source code directory
 * `App.css`: Global CSS styles
 * `App.jsx`: Main application component
 * `assets/`: Directory for project assets
 * `components/`: Directory for reusable components
 * `hooks/`: Directory for custom hooks
 * `index.css`: Global CSS styles
 * `main.jsx`: Main application entry point
 * `pages/`: Directory for page components
* `vite.config.js`: Vite configuration file

## Scripts

The following scripts are defined in `package.json`:

* `"dev": "vite"`
* `"build": "vite build"`
* `"lint": "eslint ."`
* `"preview": "vite preview"`
